# Laravel Backend API

This folder contains the Laravel-based API that powers the authentication flows for Atlas IT Solutions.

## Getting started
1. Install dependencies: `composer install`
2. Duplicate `.env.example` to `.env` and configure your database + `APP_URL`.
3. Generate the application key: `php artisan key:generate`
4. Run the migrations: `php artisan migrate`
5. Start the app: `php artisan serve`
